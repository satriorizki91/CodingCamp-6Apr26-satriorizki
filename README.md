# Productivity Dashboard

A browser-based productivity dashboard with four essential widgets: greeting display, focus timer, task manager, and quick links.

## Features

- **Greeting Widget**: Displays current time, date, and time-based greeting
- **Focus Timer**: 25-minute countdown timer for focused work sessions
- **Task List**: Manage your to-do items with completion tracking
- **Quick Links**: Save and access your favorite websites

## How to Use

1. Open `index.html` in your web browser
2. All data is stored locally in your browser (no backend required)
3. Your tasks and links will persist between sessions

## Testing

Run the test suite:
```bash
npm test
```

Run tests in watch mode:
```bash
npm run test:watch
```

## Manual Testing

Open `test-dashboard.html` in your browser to verify the Dashboard integration.
